주요 기능

기능설명

  • 주행 기능

주요 기능

기능설명

  • 장애물 회피

주요 기능

기능설명

  • 압축 기능

주요 기능

기능설명

  • 쓰레기 분류
  • 카메라에 실시간으로 인식되는 쓰레기를 분리수거 가능 여부에 따라 일반 쓰레기와 재활용 쓰레기로 이진 분류를 진행합니다.

주요 기능

기능설명

  • 쓰레기 인식
  • 커스텀 데이터셋으로 학습시킨 결과 파일을 이용하여 쓰레기통에 투입된 쓰레기를 실시간으로 인식합니다.

NH투자증권 관심그룹 플랫폼 업그레이드

2026.01 ~ 2026.06 (5개월)

C Redis Linux Shared Memory

프로젝트 소개

대량 동시 조회 환경에서도 빠른 응답속도를 제공하기 위해 Redis 기반 투자정보 조회 플랫폼을 구축하여 관심그룹 및 Mini Chart 서비스 개발

주요 성과

  • DB 기반 조회 구조를 Redis 기반으로 전환하여 조회 응답시간 약 92% 단축 (0.227초 → 0.018초)
  • Redis 기반 플랫폼 구축을 통해 대량 동시 조회 환경에서도 안정적인 응답속도 제공
  • Redis와 Shared Memory를 연계한 데이터 복구 체계를 구축하여 운영 안정성 향상

수행 업무

  • Redis 기반 관심그룹 조회 서비스 개발
  • Mini Chart(Sparkline) 조회 서비스 추가
  • 실시간 시세 데이터 처리
  • Redis 공통(Util) 라이브러리 개발 및 공통 기능 모듈화
  • 데이터 복구 프로세스 개발
  • 만료 데이터 정리 Batch 개발
  • 데이터 유실 검증 및 운영 모니터링 프로세스 개발
